Several files in this repository are managed by Git LFS. In order to download these files, the following software installation is required:
- Git - https://git-scm.com/

Once the prerequisites are met, take the following steps to download the files:
1) Open a command prompt
2) Change your current directory to your desired download destination
3) Enter the following command:
   git clone https://github.com/epri-dev/SCICHEM.git
4) The entire download will be approximately 2.62 (or 5.24 GB with the hidden .git folder included) and will take at least several minutes to download depending on your connection
