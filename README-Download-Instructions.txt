Several files in this repository are managed by Git LFS. In order to download these files, the following software installations are required:
- Git - https://git-scm.com/
- Git LFS - https://git-lfs.github.com/

Once the prerequisites are met, take the following steps to download the files:
1) Open a command prompt
2) Change your current directory to your desired download destination
3) Enter the following command:
   git clone https://github.com/epri-dev/SCICHEM.git
4) The entire download will be approximately 5.44 GB and will take at least several minutes to download depending on your connection
